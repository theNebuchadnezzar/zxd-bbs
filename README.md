# zxd-bbs
个人论坛
地址：http://www.zhaoxingdong.com/

样式使用的是CNode社区(https://cnodejs.org/)的样式。
实现了用户注册、登录、浏览帖子、发帖、头像上传等功能。帖子页面支持markdown语法。
